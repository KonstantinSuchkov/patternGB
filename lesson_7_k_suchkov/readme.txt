файл run_api.py - запуск фреймворка
На http://127.0.0.1:8080/examples весь функционал.
Проект - сайт детского садика.
Есть группы - полного дня и подготовительные.
Есть пользователи - преподаватели, родители, дети.
Добавлено:
Подключена БД (создается при первом запуске).
В БД есть две таблицы - users (пользователи - учителя, родители, дети) и parenthood (для связи родитель-ребенок).
При создании нового пользователя идет запись в БД.
Пользователя ребенка может создать только родитель на своей странице (user_detail), при этом также создается запись в таблице parenthood.
На странице родителя отображаются дети этого родителя. Доступно удаление пользователя.
